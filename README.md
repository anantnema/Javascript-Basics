Pyramid Pattern.js:

It will form a pyramid up or down with the word 'anantnema' in a repeated fashion.

    first line read left to right

    second line read right to left

    third line read left to right and so on.

  pyramid up:
  
           a            
          nan           
         tnema         
        entnana

  pyramid down:
  
        anantne        
         anaam         
          ntn
